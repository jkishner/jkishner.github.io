Send Selected Headline to Small Picture Reader

Tweet This Post

The following is an adaption of Dave Winer's script to generate a Reader macro.

How It Works

  • Place your cursor on the headline that you want to share in Small Picture Reader. (All of its subs -- or "children" -- will be included.)

  • Choose this script from the menu.

  • A new window will pop open -- Small Picture Reader -- including only that headline (not the entire outline).

  • This new OPML file is saved in a /snippets subfolder in Dropbox. Do not delete it.

Visit the Fargo scripting page in the docs to learn how to install this script, or watch this video.
